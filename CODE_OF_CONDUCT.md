# Code of Conduct

## Our Commitment

We want MediGuard to be a respectful, professional, and constructive project for everyone who participates. Contributors, maintainers, and community members should help create an environment that is welcoming, thoughtful, and safe.

## Expected Behavior

- Be respectful in language and tone.
- Assume good intent while still giving honest technical feedback.
- Focus criticism on ideas, implementation, and outcomes, not people.
- Share context clearly so others can collaborate effectively.
- Be generous with credit and careful with blame.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation
- Personal attacks or degrading language
- Bad-faith trolling or deliberate disruption
- Sharing private or sensitive information without permission
- Repeatedly ignoring project boundaries, review guidance, or community expectations

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies within repository spaces and any project-linked collaboration channels where people represent or participate in the MediGuard project.

## Reporting

If you experience or witness unacceptable behavior, report it privately through the channel described in [SUPPORT.md](SUPPORT.md). Reports will be reviewed respectfully and handled as appropriately as possible.
